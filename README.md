# 🍽️ Lumière Prestige - Restaurant Management System

**Lumière Prestige** is a sophisticated, full-stack digital solution crafted for the **Royal Gastronomy** sector. This platform streamlines elite dining operations, from elegant customer-facing landing pages to robust back-end administrative controls.

---

### 🚀 Project Overview
Lumière Prestige serves as a bridge between luxury hospitality and modern technology. The system is built to replace traditional manual workflows with a high-performance digital environment.

* **Premium Branding**: A high-contrast, elegant interface utilizing deep navy and gold accents for a "Prestige" look.
* **Dual-Portal Architecture**: Dedicated access points for public reservations ("Secure a Table") and internal staff management ("Admin Portal").
* **End-to-End Workflow**: Manages the entire lifecycle of a dining experience, from curated menu browsing to final table checkout.

---

### 🛠️ Technology Stack
* **Frontend**: React.js (Vite-powered)
* **Backend**: Node.js & Express.js
* **Database**: MongoDB (NoSQL)
* **Styling**: Tailwind CSS
* **State Management**: React Context API
* **Animations**: AOS (Animate On Scroll)

---

### ✨ Main Features
* **Elegant Landing Page**: A visually stunning homepage featuring "The Collection" and "The Estate".
* **Real-time Reservations**: Secure table booking system with instant availability updates.
* **Admin Portal**: A restricted management dashboard for staff to handle orders, menu updates, and customer data.
* **Curated Menu Management**: Dynamic categorization of "Royal Gastronomy" dishes.
* **Responsive Design**: Fully optimized for a seamless experience on mobile, tablet, and desktop.

### 💻 Local Installation Guide

1.  **Clone the Repository**:
    ```bash
    git clone https://github.com/Rusmia-Rahaman-Urfa/CodeAlpha_RestaurantManagementSystem.git
    ```

2.  **Install Dependencies**:
    ```bash
    # Install client dependencies
    cd client && npm install

    # Install server dependencies
    cd ../server && npm install
    ```

3.  **Environment Setup**:
    Create a `.env` file in the `server` directory:
    ```env
    MONGODB_URI=your_mongodb_uri
    PORT=5000
    ```

4.  **Run the Project**:
    ```bash
    npm run dev
    ```
